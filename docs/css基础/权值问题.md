# 权值问题

## 选择器

- ID选择器: #id {}

- 类选择器: .class {}

- 属性选择器: a[href="www.baidu.com"] {}

- 伪类选择器: :hover {}

- 伪元素选择器: ::after {}

- 标签选择器: div {}

- 通配符选择器: * {}
