# margin问题

# 块元素

  1. 外边距都是正数,折叠结果为两者之间较大的值

  2. 外边距是一正一负,折叠结果为两者相加的和

  3. 外边距都是负数,折叠结果为两者绝对值较大的值

# 行内元素

  1. 外边距为两者相加的值

  2. 无法设置marign-top和margin-bottom

# 行内块状元素

  1. 外边距为两者相加的值

TIP

在使用行内元素和行内块状元素的时候,会产生间隔
